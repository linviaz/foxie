# MySQL 

## Adding mysql to PATH for CLI Access
 - Temporary adding:
    ```
    export PATH=$PATH:/usr/local/mysql/bin
    ```
 - Write to bash_rc:
   add the above line to the .zshrc file.


## Command line access to MySQL server:
    ```
    mysql --user=user_name --password db_name
    ```
